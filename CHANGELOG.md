# Changelog

## v0.1.0 (2026-04-13)

First release.

### Added
- Rust client for Crontinel API
- `schedule_run` — report scheduled command outcome
- `queue_processed` — report queue worker activity
- `event` — send custom events and alerts
- Environment variables: `CRONTINEL_API_KEY`, `CRONTINEL_API_URL`
